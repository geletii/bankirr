# Bankirr for macOS

Menu bar app for [bankirr.xyz](https://bankirr.xyz). Tracks wallet balances, lending, and liquidity pools via the Bankirr API.

## Install

**Terminal (pre-built binary from bankirr.xyz):**

```bash
curl -fsSL https://bankirr.xyz/install.sh | bash
```

**Build from source:**

```bash
git clone https://github.com/geletii/bankirr.git
cd bankirr
./scripts/install.sh
```
